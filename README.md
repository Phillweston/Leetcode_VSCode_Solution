# Leetcode_VSCode_Solution

## Description

该部分内容为 Phillweston 本人的在线 Leetcode 练习题，该工作空间文件夹默认为家目录的 .leetcode 文件夹

## Plugin

采用 VSCode Leetcode 插件（Leetcode 官方版）

程序运行只能通过 VSCode 内运行，并提交至国内 Leetcode 账户，使用窗口右上角的运行按钮则会报错

## Setup

参考如下教程配置 Leetcode 插件环境：

https://blog.csdn.net/weixin_44553006/article/details/105183522

建议使用最新LTS版的 Node.js 并在插件设置中正确配置其运行路径，否则账户无法登录

国内 Leetcode 登录目前采用的是输入注册时的手机号+密码实现登录
